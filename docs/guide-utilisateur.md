# Guide utilisateur

Ce guide explique les principales possibilites de l'application.

## 1) Compte et securite
- Inscription avec verifications de mot de passe (8 caracteres minimum, 1 majuscule, 1 minuscule, 1 chiffre).
- Verification par e-mail obligatoire pour activer le compte.
- Connexion classique, renvoi de l'e-mail de verification si besoin.
- Reinitialisation du mot de passe via e-mail.

## 2) Gestion des cles API
- Ajout de cles API par fournisseur (OpenAI, Gemini, Claude, Mistral).
- Les cles sont chiffreess et stockees en base.

## 3) Chat multi-fournisseurs
- Choix du fournisseur et du modele.
- Historique des conversations conserve par fil.
- Nouvelle tentative possible en cas d'erreur de generation.

## 4) Recherche dans les conversations
- Barre de recherche dans le chat (desktop + mobile).
- Navigation entre resultats avec boutons ou clavier.
  - Enter : resultat suivant
  - Shift + Enter : resultat precedent
  - Ctrl/Cmd + F : focus sur la recherche

## 5) Pieces jointes
- Support des images (notamment pour Gemini).
- Preview dans l'interface avant envoi.

## 6) Projets et fils
- Creation de projets pour organiser les conversations.
- Instructions et contexte par projet.
- Fils (threads) par projet ou en solo.

## 7) Mode sombre
- Toggle de theme (clair / sombre).
- Interface adaptee sur toutes les pages.

## 8) Confidentialite
- Les conversations sont stockees en base pour la continute des sessions.
- Les cles API sont chiffreess.
